Enumerable Methods: ArrayList.IndexOf

ArrayList is an array with a non defined number of element in it.

IndexOf is a method that searchs string or numbers and return the zero index based (or the position) of the first occurence of a letter or an object.

So for a given ArrayList

// cs
ArrayList myNumbers = new ArrayList ();
myNumbers.Add(1);
myNumbers.Add(4);
myNumbers.Add(2);
myNumbers.Add(7);
myNumbers.Add(2);
myNumbers.Add(19);

// Say we want to know were is located the first occurence of the number 2 within the list.

// We declare the object.

int theNumber = 2;
Console.WriteLine("The first occurence of " + theNumber + " is at the position" + myNumbers.IndexOf( theNumber ) + " in the list")
// cs
We can use this Method to locate an occurence from the beginning of the list, as we did, but also we can use locate an occurence between a certain point of the list to another.
// cs
int theNumber = 2;
myNumbers.IndexOf( theNumber, 5) // between the fifth number and the end of the list
myNumbers.IndexOf( theNumber, 5, 9) // between the fifth and the ninth
myNumbers.IndexOf( theNumber, 16) // if the number is outside of the list the method will return -1.

Arrays vs. Hashes. What are they?
Array = a list of element that are stored like in a table with a position automatically given from the beginning to the end.
Hashes = a list of element that aren't ordered but from groups. They have the name of a "group" called a key that is arbitrary choosen by you.
They role is to store data that will be accessed later on.
Each one have a different use and if you have a long list of data to store, that need to be ordered but without knowing the number of information you will store I would recomment to use array.

On another hand, hashes are way more secured as you need the key to access to the information, in the way that some password method use the hashtables to confirm that a key correspond to an element.

A good strategy would always be to ask yourself. What am I gonna use this list for? Do I know in advance the number of data I will store? What kind of data will it be? Do I need to secure this data?
